Mac & Windows Online Help

Online Help for Windows

Online scanners are cloud-based apps that search for virus/malware infections without installing a full program on the computer.  Almost all will work with Safari, Edge, Chrome and the Firefox browser. In most cases a single file will be downloaded to your Downloads folder, which you will open and run. Note, however, that the scan can take a long time, even with a fast computer and internet connection. Be sure to enable the "Search for Rootkits" option in Settings (if available) before you start the scan.

ESETF-SecurePandaNorton

Online scanners are practical when you are looking for verification that a problem has been fixed or that a computer is infected. Unfortunately, only some online scanners will remove malware. These tools, however, can remove any malware they find, including rootkits, particularly Norton's online scanner. Most importantly, since two antivirus programs should not be installed on a computer at the same time, online scanners are safe to use with your current antivirus program running in the background, including Microsoft's Windows Defender.  None of the online scanners above install a full program meant for daily use.

Online Help for Macs

Mac computers are not immune to viruses or malware, it's just that since there are so few of them compared the number of Windows computers, there is much less malware written to attack the Mac OS. Currently on desktop and notebooks, Windows claims a 71% market share while Macs have 16%.  All newer versions of MacOS have antimalware and anti-virus software built-in and it updates automatically and runs continuously.  It is still a good idea to have antivirus and malware protection for your Mac if you visit questionable websites. If you suspect a threat on your Mac (OS X -10.6 or later.), try downloading and running two or three of the products below to remove any infections.

Online Help for Routers

Consumer router and ISP gateway router security can be poor. Attackers take advantage of lackadaisical ISPs and are able to attack a large number of routers at a time. Use the links below to check if the Domain Name Server (DNS) address(es) on the router have been compromised. This setting can be changed manually if necessary. Domain Name Servers are the Internet's equivalent of a phone book. They maintain a directory of domain names (like https://google.com) and translate them to Internet Protocol (IP) addresses. For Google, one of those addresses is 142.251.35.174
