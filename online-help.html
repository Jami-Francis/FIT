Mac & Windows Online Help

Online Help for Windows

Online scanners are Cloud based apps that search for virus/malware infections without installing a full program on the computer.  Almost all will work within Safari, Edge, Chrome and the Firefox browser.  However, the scan can take a long time, even with a fast computer and internet connection.

ESET Online ScannerF-Secure Online ScannerPanda's Cloud Cleaner

Scanners are practical when you are looking for verification that a problem has been fixed or that a computer is infected. Unfortunately, only some scanners will remove malware, however those listed above can remove all malware it finds. Most importantly, since two antivirus programs should not be installed on a computer at the same time, online scanners are safe to use with your current antivirus and / or antispyware program running in the background.  Any of these three are a good compliment to Microsoft's Defender AV app.

Online Help for Macs

Mac computers are not immune to viruses or malware, it's just that since there are so few of them compared the number of Windows computers, there is much less malware written to attack the Mac OS. All newer versions of MacOS have antimalware and anti-virus software built-in and it updates automatically and runs continuously.  It is still a good idea to have antivirus and malware protection for your Mac if you visit questionable websites. If you suspect a threat on your Mac (OS X -10.6 or later.), try downloading and running two or three of the products below to remove any infections.

Online Help for Routers

Consumer router and ISP gateway router security can be poor. Attackers take advantage of lackadaisical manufacturers and are able to attack a large number of routers at a time. Use the links below to check if the Domain Name Server (DNS) address(es) on the router have been compromised. This setting can be changed manually if necessary. Domain Name Servers are the Internet's equivalent of a phone book. They maintain a directory of domain names (like https://google.com) and translate them to Internet Protocol (IP) addresses. For Google, one of those addresses is 142.251.35.174
