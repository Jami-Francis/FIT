Disable AI in Chrome & Firefox

Firefox

You can disable Firefox’s AI features using the built-in AI Controls or the advanced configuration menu.
 
Method 1: Use the AI Controls (Easiest)
  1. Click the Menu button (three horizontal lines) in the top-right corner and select Settings.
  2. Select AI Controls on the left sidebar.
  3. Toggle on Block AI enhancements.
  4. If you want to disable specific AI features (like AI tab group suggestions or link previews) individually, use the drop-down menus on this page to set them to Blocked.
Method 2: Force-disable via about:config (Advanced)
If you want to ensure all background AI hooks are completely disabled, you can use Firefox's advanced settings:
  1. Type about:config in the address bar and press Enter.
  2. Click Accept the Risk and Continue.
  3. Search for the following preferences and double-click to set them to false:
    • browser.ml.enable
    • browser.ml.chat.enabled
    • browser.ml.linkPreview.enabled
    • browser.tabs.groups.smart.enabled
    • extensions.ml.enabled
  4. Search for browser.ai.control.default and change its value to blocked.
Note: If any local AI models were previously downloaded to your computer, they will be automatically purged when these features are disabled
